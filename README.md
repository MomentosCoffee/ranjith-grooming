# Ranjith — Web 🐾

Web de ejemplo (REMAKE) para **Ranjith** — Dog Grooming · Training · Show Handling, Calgary AB.
HTML/CSS/JS puro, una página, bilingüe EN/ES, mobile-first. Marca dorado #FCB305 + negro.

## 📷 Subir fotos (auto-detectan, sin tocar código) → assets/img/
| Archivo | Qué va |
|---|---|
| `hero.jpg` | Hero (perro estrella / Doberman) |
| `grooming.jpg` `training.jpg` `handling.jpg` | Las 3 cards de servicios |
| `about.jpg` | Ranjith con un perro |
| `g1.jpg` `g2.jpg` | Galería (tus triunfos / transformaciones) |
| `logo-white.png` | Logo (ya puesto, dorado transparente) |

Sirve jpg/png/webp. Las que no reemplaces quedan con stock.

## ✏️ CONFIG (arriba del `<script>`)
- `PHONE` → número real de Ranjith (hoy placeholder 403-000-0000)
- `IG` → link de Instagram real

## 🚀 Local / Deploy
`python3 -m http.server 3456` → http://localhost:3456
Deploy: crear repo `ranjith-web` (MomentosCoffee) → push → Import en Vercel.
